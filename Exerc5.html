    -Categoria do Curso

    -Descricao do curso

    Este curso apresenta conceitos basicos de web e mostra as parteas e funcoes das tags
    em um documento HTML. Ensina, tambem, a formatar uma pagina, inserir imagens e
    muito mais! Tudo isso, com explicacoes simples e de facil compreensao

    -Estrutura do curso

      1. Modulo 1 - Introducao

      2. Modulo 2 - HTML e a web

      3. Modulo 3 - Criando um documento HTML

      4. Modulo 4 - Formatando um documento HTML

      5. Modulo 5 - Criando itens

      6. Modulo 6 - Inserindo imagens

      7. Modulo 7 - Criando links

      8. Modulo 8 - Combinando tags

      9. Modulo 9 - Usando imagens mapeadas
